Plant Phenotyping: Image Analysis of Pear Rootstocks

Washington State University | Jun 2022 - Jul 2022

Developed a Python workflow to automate phenotyping data collection for pear rootstocks, achieving 97% data accuracy and increasing efficiency by 60%.

Project Overview

This project focused on analyzing pear rootstocks for key architectural traits to support breeding programs. Using Python and PlantCV, I automated the data collection process for phenotypic traits like stem thickness and leaf area.

Presented findings at the Summer Research Symposium, contributing to agricultural research for more resilient pear varieties.

Skills Utilized

Data Analysis, Process Automation, Python, PlantCV
